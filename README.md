# CoDeIn-usingjs
A project to compile, deploy and interact with a smart contract using js.

## Prerequisites
- Node.js
- npm

## Installation
1. Clone the repository
```bash
git clone https://github.com/your-repo/codein-usingjs.git
```
2. Install dependencies
```bash
npm install
```
3. Run the project
```bash
node main.js
```
